Meta's Superintelligence Lab has launched **Muse Spark**, the first multimodal model in the Avocado family. The model is accessible via a website and official app, with a closed API testing phase underway for developers.

Muse Spark features visual chain-of-thought reasoning, external tool integration, and agent orchestration, highlighted by its unique "Contemplating Mode" where multiple AI agents analyze complex tasks simultaneously. Independent tests rank Muse Spark in the top 5 AI models, trailing only GPT-5.4 and Gemini 3.1 Pro.

The creators report strong performance in visual STEM tasks and medical analytics but note challenges in coding and long multi-step scenarios.
