title: "GitHub Copilot Inserts Ads into Pull Requests, Sparking Developer Backlash"
date: 2026-03-31
tags: news
categories: news
---

Independent developer Zach Manson discovered that GitHub Copilot inserted advertising text into pull requests while auto-correcting typos. This ad promoted the Raycast utility for launching Copilot on macOS and Windows, with tens of thousands of PRs reportedly affected.

A Copilot representative confirmed the incident, calling it a mistake and quickly disabling the feature. However, the event has seriously damaged trust within the developer community, prompting some users to migrate private repositories from GitHub to alternative platforms like Forgejo, Gitea, Codeberg, and self-hosted solutions.
