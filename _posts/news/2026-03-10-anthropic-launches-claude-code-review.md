Anthropic has introduced **Claude Code Review**, an AI-powered tool designed to detect bugs in pull requests. Currently available in preview for Team and Enterprise corporate users, the tool automatically activates AI agents upon PR opening to analyze and prioritize vulnerabilities by severity.

The review process typically takes about 20 minutes, with pricing based on token usage ranging from $15 to $25 per pull request. Internal tests show a significant increase in meaningful PR comments, from 16% to 54%, after the tool's implementation.
