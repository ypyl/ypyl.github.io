Microsoft and GitHub rolled out a wave of developer-focused releases, headlined by the new **GitHub Copilot desktop app** in preview — an agent-native experience — and **Frontier Tuning**, a private-preview capability for fine-tuning agents to enterprise-specific workflows. On the Windows side, developers are getting native **Coreutils** (Linux-style CLI utilities), Linux container support through WSL, and a new **Intelligent Terminal** that shares context with AI agents. Microsoft also previewed **Project Solara**, a platform for AI-agent-powered devices, showing two reference designs: a desktop hub with facial recognition and a wearable badge equipped with a camera and real-time conversation transcription.
