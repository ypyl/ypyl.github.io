Nous Research has released a preview of **Hermes Desktop**, an MIT-licensed application for deploying AI agents on **Windows, macOS, and Linux**. The tool operates across terminal, email, and messengers — planning tasks in natural language and maintaining persistent memory to preserve context across projects. Its architecture supports **sub-agent delegation**: Hermes spawns agent instances with isolated terminals for Python scripting, web search, image generation, and speech synthesis.

Code execution is sandboxed across five environments — local, **Docker**, **SSH**, **Singularity**, and **Modal** — while LLM requests are routed through [Nous Portal](https://portal.nousresearch.com/manage-subscription), which offers over 300 models including free-tier options.
