Microsoft has expanded its Sovereign Cloud offerings to enhance governance, productivity, and support for large AI models that can run securely even when fully disconnected from the cloud. The updates include Azure Local disconnected operations for mission-critical infrastructure, Microsoft 365 Local for core productivity workloads within sovereign boundaries, and Foundry Local, which enables large AI models to run on-premises with modern infrastructure from partners like NVIDIA. This unified Sovereign Private Cloud solution ensures continuous operations, data protection, and consistent governance across connected and disconnected environments.
