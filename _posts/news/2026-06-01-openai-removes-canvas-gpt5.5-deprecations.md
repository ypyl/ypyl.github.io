date: 2026-06-01
tags: [news]
categories: news
---

OpenAI has **removed the Canvas feature** from both GPT-5.5 Instant and GPT-5.5 Thinking, with Canvas elements now rendering inline in the main chat window. Alongside this change, GPT-5.5 Instant was updated to produce **fewer long lists and more natural, readable responses**. Paid users can still access the classic Canvas by manually switching to older model versions. OpenAI also announced deprecation timelines: **GPT-4.5 will be removed from the web interface on June 27** (already gone from the API), and **o3 will leave the web version on August 26, 2026**, though API access will remain.
