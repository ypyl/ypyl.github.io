Apple has introduced its **third generation of Apple Foundation Models (AFM 3)** — a family of five models developed in collaboration with Google, spanning on-device to server-based deployments. The standout is **AFM 3 Core Advanced**, a 20-billion-parameter model that runs entirely on iPhone 17 Pro by storing the full model in flash memory and loading only 1–4 billion "expert" parameters into active DRAM per request, enabling expressive voices and higher-accuracy dictation without leaving the device. The lineup also includes **AFM 3 Cloud Pro**, Apple's most capable server model, now running on **NVIDIA GPUs** in Google Cloud via an expanded Private Cloud Compute infrastructure.
