title: "Anthropic Reverses Agent SDK Pay-As-You-Go Plan"
date: 2026-06-17
tags: news
categories: news
---

Anthropic has cancelled a billing overhaul for the **Claude Agent SDK**, interactive mode, and third-party integrations that was set to take effect June 15, 2026. The plan would have moved these tools outside standard subscriptions with monthly credits ($20 for Pro, $200 for Enterprise) and token-based overage charges at standard API rates—instead they will continue using shared plan limits. Anthropic cited a need to align pricing with real-world usage patterns, though some analysts link the reversal to **OpenAI preparing steep API price cuts** to compete for users. The company says it will revisit separate agent billing after reducing its own base token costs.
