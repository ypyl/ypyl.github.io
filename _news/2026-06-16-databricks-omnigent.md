Databricks has released **Omnigent** as an open-source alpha under Apache 2.0—a meta-harness that unifies AI agents like Claude Code and custom solutions through a shared API. It moves model control from prompts to an OS-level sandbox, enabling financial limits on API usage and trigger-based rules (e.g., requiring human approval on `git push` if the agent downloaded a new network package beforehand). The platform supports multi-user sessions and allows sharing active context via a link for real-time collaborative review and task adjustment.
