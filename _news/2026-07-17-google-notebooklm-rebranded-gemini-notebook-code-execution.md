Google has rebranded **NotebookLM** as **Gemini Notebook**, adding native code execution capabilities. Each notebook now includes an isolated cloud environment for writing and running scripts, enabling users to analyze data directly from uploaded sources without exporting to external tools. While remaining a standalone product, it is now more deeply integrated into Google's ecosystem — notebooks are created and synced within the Gemini app and will later appear in Google Search's AI results. Code execution is currently available for **Google AI Ultra** subscribers and **Workspace** enterprise customers, with rollout to **Pro** tier users expected within weeks.
