date: 2013-06-01
categories: javascript programming
---
If you use bootstrap and recaptcha, you may face a problem like wrong padding or margin.

You should add next css style to yours:

{% highlight css %}
body{ line-height:1}
{% endhighlight %}

And all will be OK ! Enjoy!
