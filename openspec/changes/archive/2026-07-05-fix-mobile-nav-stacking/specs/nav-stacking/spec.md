No new or modified requirements. This change fixes a CSS z-index bug where the mobile navigation dropdown renders behind the news carousel cards. The `.site-header` and `.news-carousel-views` both use `position: relative` with `z-index: auto`, and CSS stacks later-DOM elements higher. Adding `z-index: 10` to `.site-header` lifts the header and its nav dropdown above all main content — no spec-level requirement changes.
